# Simple Drag-and-Drop Blog Builder

This is a basic implementation of a drag-and-drop interface for creating a simple blog using HTML, CSS, and JavaScript.

## Table of Contents

- [Features](#features)
- [Getting Started](#getting-started)
- [Usage](#usage)
- [Screenshots](#screenshots)
- [Contributing](#contributing)
- [License](#license)

## Features

- Drag-and-drop elements (Text and Image) onto a blog canvas.
- Save the blog content (simulated with an alert).
- Minimalistic user interface.
